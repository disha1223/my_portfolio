# 🌸 Disha Portfolio

A modern, responsive developer portfolio built with **React**, **Vite**, and **Tailwind CSS**. The portfolio showcases my projects, technical skills, experience, resume, and contact information with a clean Apple-inspired design and soft pastel aesthetics.

## ✨ Features

- 🎨 Modern responsive UI
- 💻 Apple-inspired minimal design
- 📱 Mobile-friendly layout
- 📂 Project showcase
- 🧠 Skills section
- 💼 Experience timeline
- 📄 Resume download
- 📬 Contact section
- ⚡ Fast performance with Vite
- 🎀 Smooth animations and transitions

---

## 🛠️ Tech Stack

- React
- Vite
- Tailwind CSS
- JavaScript (ES6+)
- HTML5
- CSS3

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/your-username/your-repository.git
```

### Navigate to the project

```bash
cd your-repository
```

### Install dependencies

```bash
npm install
```

### Start development server

```bash
npm run dev
```

Open:

```
http://localhost:5173
```

---

## 📦 Build for Production

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
